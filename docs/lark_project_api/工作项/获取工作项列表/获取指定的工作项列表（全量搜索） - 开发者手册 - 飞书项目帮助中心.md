该接口用于在指定空间，搜索符合筛选条件的工作项实例，支持字段按需查询。对应的权限申请在 权限管理-工作项 分类下，相关功能介绍详见 权限管理 。

请求说明

- 请求方式：POST

- 请求地址：/open_api/view_search/universal_search

请求参数

Body

| 参数                    | 类型           | 是否必须 | 说明                                                                                                                                                                                                                                                |
| --------------------- | ------------ | ---- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| project_key          | string       | 必须   | 飞书项目空间的唯一标识，在飞书项目空间双击空间名称即可获取。                                                                                                                                                                                                                    |
| work_item_type_key | string       | 必须   | 工作项类型，通过 [获取空间下工作项类型](https://project.feishu.cn/b/helpcenter/1p8d7djs/3pjp854w) 接口获取。                                                                                                                                                             |
| search_group         | object       | 必须   | 筛选条件（支持上限最多50个），用于组合条件搜索时使用。遵循 [SearchGroup](https://meego-hc.larkoffice.com/b/helpcenter/2.0.0/1p8d7djs/1x1d372l#a889b622) 结构规范，可参考 [搜索参数格式及常用示例](https://meego-hc.larkoffice.com/b/helpcenter/2.0.0/1p8d7djs/1l8il0l6) 。                        |
| pagination            | object       | 可选   | 分页                                                                                                                                                                                                                                                |
| field_selected       | list<string> | 可选   | **field_selected查询参数说明**  1. **字段选择机制** ：支持按需查询，用户可自定义指定需返回的字段集合，该参数为非必选项。如未指定，系统默认仅返回工作项ID。 2. **字段数量限制** ：当前系统设定单次查询最多可返回20个字段。 3. **字段格式规范** ：具体字段格式定义及属性解析规则，请参阅 [字段与属性解析格式](https://project.feishu.cn/b/helpcenter/1p8d7djs/1tj6ggll) 说明文档。 |

返回参数

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| pagination | object | 分页信息 |
| datas | list<object> | 工作项详情新模型 |
| err | object | 请求成功时返回空值，请求失败时   返回实际错误信息。 |
| err_msg | string | 请求成功时返回空值，请求失败时   返回实际错误信息。 |
| err_code | int32 | 请求成功时返回0，请求失败返回   实际错误码。 |

请求示例

获取指定工作项

带 search_after 分页配置

```json
{
  "project_key": "66542b5xxxxxxxxxxxxf15449", // 空间key
  "work_item_type_key": "story", // 工作项类型
  "search_group": { // 筛选条件
    "conjunction": "AND",
    "search_params": [
      {
        "param_key": "created_at",
        "value": 1654064482000,
        "operator": ">"
      }
    ]
  },
  "field_selected": [ // 查询参数
    "work_item_id",
    "created_at",
    "name",
    "project_key"
  ],
  "pagination": { // 分页参数
    "page_size": 6
  }
}
```
