## 🔐 鉴权依赖说明
### 凭证类型与适用场景
| 凭证类型 | 获取方式 | 适用 API | Header 要求 |
|----------|----------|----------|-------------|
| **tenant_access_token** | app_id + app_secret | 大部分 API | `Authorization: Bearer xxx` |
| **user_access_token** | OAuth 授权 | 需要用户身份的操作 | `Authorization: Bearer xxx` + `X-USER-KEY` |
### 凭证获取流程
```
app_id + app_secret
        │
        ▼
┌───────────────────┐     ┌───────────────────┐
│ POST /auth/v3/    │ ──► │ tenant_access_   │
│ tenant_access_    │     │ token (2小时有效) │
│ token/internal    │     └───────────────────┘
└───────────────────┘
        │
        │ (需要用户身份时)
        ▼
授权页面 ──► code ──► /oauth/v3/access_token
                              │
                              ▼
                    ┌───────────────────┐
                    │ user_access_      │
                    │ token +          │
                    │ X-USER-KEY       │
                    └───────────────────┘
```
### 所有 API 调用都需要
| Header | 说明 |
|--------|------|
| `Authorization` | Bearer token (必填) |
| `Content-Type` | application/json (必填) |
| `X-USER-KEY` | user_key (使用 user_access_token 时必填) |

    │
    ├──► 工作项类型 (work_item_type_key)
    │         │
    │         ├──► 工作项搜索
    │         │
    │         ├──► 工作项创建/查询/更新/删除
    │         │
    │         ├──► 字段配置
    │         │
    │         └──► 流程模板

    └──► 用户组

    │     └──► 子任务

## 📌 关键依赖参数
| 参数 | 来源 API | 被依赖 API |
|------|----------|------------|
| `project_key` | 空间列表/空间详情 | 所有工作项 API、配置 API |
| `work_item_type_key` | 获取空间下工作项类型 | 工作项 CRUD、搜索、配置 |
| `work_item_id` | 创建/搜索工作项 | 详情、更新、删除、流程、子任务、附件、工时等 |
| `node_id` | 工作流详情 | 节点更新、节点完成、子任务 |
| `view_id` | 视图列表 | 视图操作、获取视图下工作项 |
| `field_key` | 获取字段信息 | 附件上传、字段更新 |
| `template_id` | 获取流程模板列表 | 流程模板详情、更新、删除 |
| `task_id` | 创建/查询子任务 | 子任务更新、删除、完成 |

## 🚦 API 调用规则
### 必先获取的参数
| 场景 | 必须先获取 | 然后调用 |
|------|-----------|----------|
| 创建工作项 | `project_key` + `work_item_type_key` | 创建工作项 |
| 查询工作项 | `project_key` + `work_item_type_key` | 搜索/查询 |
| 管理流程 | `work_item_id` | 流程详情、节点操作 |
| 管理子任务 | `work_item_id` + `node_id` | 子任务 CRUD |
| 上传附件 | `work_item_id` + `field_key` | 附件上传 |
| 工时登记 | `work_item_id` | 工时 CRUD |

## 📋 API 分类依赖矩阵
| 分类 | 依赖 | 被依赖 |
|------|------|--------|
| **鉴权层** | app_id + app_secret | 所有 API 的前置条件 |
| **用户与空间** | 鉴权 token | 被其他所有 API 依赖 |
| **配置 API** | 空间 + 鉴权 token | 被工作项 API 依赖 |
| **工作项搜索** | 空间 + 工作项类型 + 鉴权 token | 为其他 API 提供 work_item_id |
| **工作项 CRUD** | 空间 + 工作项类型 + 鉴权 token | 核心数据操作 |
| **工作项详情** | work_item_id + 鉴权 token | 流程/子任务/附件前置 |
| **流程与节点** | work_item_id + 鉴权 token | 依赖工作项存在 |
| **子任务** | work_item_id + node_id + 鉴权 token | 依赖流程节点 |
| **附件** | work_item_id + field_key + 鉴权 token | 依赖工作项和字段 |
| **视图** | space + work_item_type_key + 鉴权 token | 提供数据展示 |
| **工时** | work_item_id + 鉴权 token | 依赖工作项 |
| **空间关联** | work_item_id + 鉴权 token | 依赖工作项关联 |
