搜索参数是用于构建工作项筛选条件的核心元素，通过指定参数键、操作符和值，可精准筛选符合需求的工作项。

## SearchGroup 筛选组合

筛选器中一行是一个 SearchParam，一组为一个 SearchGroup。

![SearchGroup示意图](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/181a710102ce4dc2a36bfb25ce428255~tplv-goo7wpa0wc-quality:q75.awebp)

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| search_params | list<SearchParam> | 筛选条件列表 |
| conjunction | string | 组合关系，枚举 AND/OR |
| search_groups | list<SearchGroup> | 嵌套的筛选组合 |

## SearchParam 筛选参数

> **说明**：筛选参数不支持字段类型为公式计算的字段。

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| param_key | string | 筛选条件参数 key |
| value | interface{} | 搜索字段值 |
| operator | string | 操作符类型 |
| pre_operator | string | 前置操作符，目前仅用于复合字段搜索场景 |
| value_search_groups | SearchGroup | 复合字段子字段搜索组 |

## 操作符枚举值

| 操作符 | 说明 |
| --- | --- |
| `~` | 匹配 |
| `!~` | 不匹配 |
| `=` | 等于 |
| `!=` | 不等于 |
| `<` | 小于 |
| `>` | 大于 |
| `<=` | 小于等于 |
| `>=` | 大于等于 |
| `HAS ANY OF` | 存在选项属于 |
| `HAS NONE OF` | 全部选项均不属于 |
| `IS NULL` | 为空 |
| `IS NOT NULL` | 不为空 |
| `CONTAINS` | 包含 |
| `NOT CONTAINS` | 不包含 |
| `MEET` | 满足 |
| `NOT MEET` | 不满足 |

## 前置操作符枚举值

| 前置操作符 | 说明 |
| --- | --- |
| `EVERY` | 每一组 |
| `ANY` | 存在一组 |

## 固定参数及取值

> **说明**：如果操作符选的是「为空」或「不为空」，不校验参数值（除流程节点时间、节点负责人、角色人员会校验参数值）。

| 参数名 | 参数 key | 操作符可取值 | 参数值类型 | 说明 |
| --- | --- | --- | --- | --- |
| 进行中节点 | current_nodes | `=`、`!=`、`HAS ANY OF`、`HAS NONE OF`、`CONTAINS`、`NOT CONTAINS` | list<string> | 节点名称列表 |
| 流程节点 | all_states | `=`、`!=`、`HAS ANY OF`、`HAS NONE OF`、`CONTAINS`、`NOT CONTAINS` | list<string> | 节点名称列表 |
| 流程节点时间 | feature_state_time | `<`、`>`、`<=`、`>=`、`IS NULL`、`IS NOT NULL` | object | 见下方示例 |
| 全部人员 | people | `=`、`!=`、`HAS ANY OF`、`HAS NONE OF`、`IS NULL`、`IS NOT NULL`、`CONTAINS`、`NOT CONTAINS` | list<string> | user_key 列表 |
| 创建时间 | created_at | `=`、`!=`、`<`、`>`、`<=`、`>=` | int64 | 毫秒时间戳 |
| 指定节点负责人 | node_owners | `=`、`!=`、`HAS ANY OF`、`HAS NONE OF`、`IS NULL`、`IS NOT NULL`、`CONTAINS`、`NOT CONTAINS` | list<object> | 见下方示例 |
| 工作项 ID | work_item_id | `=`、`!=`、`<`、`>`、`<=`、`>=`、`HAS ANY OF`、`HAS NONE OF` | list<int64> | - |
| 工作项状态 | work_item_status | `=`、`!=`、`HAS ANY OF`、`HAS NONE OF` | list<string> | 从「获取字段信息」接口获取 |
| 模板 ID | template_id | `=`、`!=`、`HAS ANY OF`、`HAS NONE OF`、`IS NULL`、`IS NOT NULL` | list<int64> | - |
| 业务线 | business | `=`、`!=`、`HAS ANY OF`、`HAS NONE OF`、`IS NULL`、`IS NOT NULL` | list<string> | - |
| 角色人员 | role_owners | `=`、`!=`、`HAS ANY OF`、`HAS NONE OF`、`IS NULL`、`IS NOT NULL`、`CONTAINS`、`NOT CONTAINS` | list<object> | 见下方示例 |

### 流程节点时间参数格式

```json
{
  "state_name": "Android开发估分",      // 节点名称
  "state_timestamp": 1702310399000,     // 节点筛选时间戳
  "state_condition": 1                 // 0=开始，1=结束
}
```

### 节点负责人参数格式

```json
[
  {
    "state_name": "开始",
    "owners": ["user_key"]
  }
]
```

### 角色人员参数格式

```json
[
  {
    "role": "role_id",
    "owners": ["user_key"]
  }
]
```

## 自定义类型及取值

> **📌 说明**：如果操作符选的是「为空/不为空」，不校验字段值，只校验当前字段类型是否支持该操作符。

| 参数类型 | 字段类型 Key | 操作符可取值 | 字段值类型 | 说明 |
| --- | --- | --- | --- | --- |
| 单行文本/多行文本 | text | `~`、`!~`、`=`、`!=`、`IS NULL`、`IS NOT NULL` | string | 前后模糊匹配 |
| 数字 | number | `=`、`!=`、`<`、`>`、`<=`、`>=`、`IS NULL`、`IS NOT NULL` | float64 | - |
| 链接 | link | `~`、`!~`、`=`、`!=`、`IS NULL`、`IS NOT NULL` | string | - |
| 开关 | bool | `=`、`!=`、`IS NULL`、`IS NOT NULL` | bool | 搜空值需使用 `IS NULL` |
| 单值系统外信号 | signal | `=`、`!=`、`HAS ANY OF`、`HAS NONE OF` | list<string> | `"undefined"`=暂无，`"null"`=进行中，`"false"`=未通过，`"true"`=已通过 |
| 单选 | select | `=`、`!=`、`HAS ANY OF`、`HAS NONE OF`、`IS NULL`、`IS NOT NULL` | list<string> | 使用选项 value 值 |
| 单选按钮 | radio | `=`、`!=`、`HAS ANY OF`、`HAS NONE OF`、`IS NULL`、`IS NOT NULL` | list<string> | - |
| 多选 | multi-select | `=`、`!=`、`HAS ANY OF`、`HAS NONE OF`、`IS NULL`、`IS NOT NULL`、`CONTAINS`、`NOT CONTAINS` | list<string> | - |
| 级联单选 | tree-select | `=`、`!=`、`HAS ANY OF`、`HAS NONE OF`、`IS NULL`、`IS NOT NULL` | list<string> | - |
| 级联多选 | tree-multi-select | `=`、`!=`、`HAS ANY OF`、`HAS NONE OF`、`IS NULL`、`IS NOT NULL` | list<string> | - |
| 单选人员 | user | `=`、`!=`、`HAS ANY OF`、`HAS NONE OF`、`IS NULL`、`IS NOT NULL` | list<string> | user_key 列表 |
| 多选人员 | multi-user | `=`、`!=`、`HAS ANY OF`、`HAS NONE OF`、`IS NULL`、`IS NOT NULL`、`CONTAINS`、`NOT CONTAINS` | list<string> | user_key 列表 |
| 富文本 | multi-text | `~`、`!~`、`IS NULL`、`IS NOT NULL` | string | - |
| 单选关联工作项 | workitem_related_select | `=`、`!=`、`HAS ANY OF`、`HAS NONE OF`、`IS NULL`、`IS NOT NULL` | list<int64> | - |
| 多选关联工作项 | workitem_related_multi_select | `=`、`!=`、`HAS ANY OF`、`HAS NONE OF`、`IS NULL`、`IS NOT NULL`、`CONTAINS`、`NOT CONTAINS` | list<int64> | - |
| 日期时间 | precise_date | `<`、`>`、`<=`、`>=`、`IS NULL`、`IS NOT NULL` | int64 | - |
| 日期 | date | `<`、`>`、`<=`、`>=`、`IS NULL`、`IS NOT NULL` | int64 | - |
| 复合字段-父字段 | compound_field | `IS NULL`、`IS NOT NULL`、`MEET`、`NOT MEET` | - | 需配合 pre_operator 使用 |
| 复合字段-子字段 | - | 支持上述除复合字段外的所有操作符 | - | - |
| 电话号码 | telephone | `~`、`!~`、`=`、`!=`、`IS NULL`、`IS NOT NULL` | string | - |
| 电子邮件 | email | `~`、`!~`、`=`、`!=`、`IS NULL`、`IS NOT NULL` | string | - |

## 常用查询示例

### 查询需求下指定状态，且创建时间在某个区间的工作项列表

```json
{
  "page_size": 50,
  "page_num": 1,
  "search_group": {
    "conjunction": "AND",
    "search_params": [
      {
        "param_key": "created_at",
        "value": 1654064482000,
        "operator": ">"
      },
      {
        "param_key": "created_at",
        "value": 1654063482000,
        "operator": "<"
      },
      {
        "param_key": "work_item_status",
        "value": ["start"],
        "operator": "="
      }
    ]
  }
}
```

### 查询需求下包含指定人员的所有工作项

```json
{
  "page_size": 50,
  "page_num": 1,
  "search_group": {
    "conjunction": "AND",
    "search_params": [
      {
        "param_key": "people",
        "value": ["XXX"],
        "operator": "HAS ANY OF"
      }
    ]
  }
}
```

### 通过需求 ID 查询指定的需求

```json
{
  "page_size": 50,
  "page_num": 1,
  "search_group": {
    "conjunction": "AND",
    "search_params": [
      {
        "param_key": "work_item_id",
        "value": [12345, 45678],
        "operator": "HAS ANY OF"
      }
    ]
  }
}
```

### 查询一个需求下关联的所有缺陷

使用 `_field_linked_story` 字段：

```json
{
  "page_size": 50,
  "page_num": 1,
  "search_group": {
    "conjunction": "AND",
    "search_params": [
      {
        "param_key": "_field_linked_story",
        "value": [12345],
        "operator": "="
      }
    ]
  }
}
```

### 复杂组合查询

查询「不存在优先级属于 P2」或者「业务线等于某个值且 PM 角色人员为指定人员」的工作项：

```json
{
  "search_group": {
    "conjunction": "OR",
    "search_params": [
      {
        "param_key": "priority",
        "value": ["2"],
        "operator": "HAS NONE OF"
      }
    ],
    "search_groups": [
      {
        "conjunction": "AND",
        "search_params": [
          {
            "param_key": "business",
            "value": ["xxx"],
            "operator": "="
          },
          {
            "param_key": "role_owners",
            "value": [
              {
                "role": "pm",
                "owners": ["xxx"]
              }
            ],
            "operator": "="
          }
        ]
      }
    ]
  }
}
```

### 复合字段数据 - 每一组子字段都等于1

```json
{
  "page_size": 50,
  "page_num": 1,
  "search_group": {
    "conjunction": "AND",
    "search_params": [
      {
        "param_key": "field_xxxxxx",
        "operator": "MEET",
        "pre_operator": "EVERY",
        "value_search_groups": {
          "conjunction": "AND",
          "search_params": [
            {
              "param_key": "field_xxxxxx",
              "operator": "=",
              "value": "1"
            }
          ]
        }
      }
    ]
  }
}
```
