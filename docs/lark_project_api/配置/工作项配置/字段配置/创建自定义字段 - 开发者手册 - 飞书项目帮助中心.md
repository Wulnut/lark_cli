该接口用于在指定工作项类型下创建一个新的自定义字段，对应的平台功能介绍详见字段配置。对应的权限申请在权限管理-工作项实例分类下，相关功能介绍详见权限管理。

注意事项

- 工作项关系字段默认数据可见范围为固定数据范围，不支持修改条件数据范围
- 附件类型字段暂不支持设置默认值，系统外信号/多值系统外信号不支持设置默认值
- 暂不支持修改字段有效性（默认有效）
- 不支持更新工作项类型、创建者、提出时间、完成时间、业务线字段
- 不支持投票字段

请求说明

- 请求方式：POST

- 请求地址：/open_api/:project_key/field/:work_item_type_key/create

请求参数

Path

| 参数 | 类型 | 是否必须 | 说明 |
| --- | --- | --- | --- |
| project_key | string | 必须 | 空间 ID（project_key），或者空间域名（simple_name）。  - 空间 ID（project_key），在飞书项目空间双击空间名称获取。 - 空间域名（simple_name），simple_name 一般在飞书项目空间 URL 中获取。例如空间 URL 为 "https://project.feishu.cn/doc/overview"，则 simple_name 为 "doc"。 |
| work_item_type_key | string | 必须 | 工作项类型，系统自动生成，可以从 [获取空间下工作项类型](https://project.feishu.cn/b/helpcenter/2.0.0/1p8d7djs/3pjp854w) 接口获取。 |

Body

| 参数 | 类型 | 是否必须 | 说明 |
| --- | --- | --- | --- |
| field_name | string | 必须 | 字段的名称，不允许与其他字段重复 |
| field_alias | string | 可选 | 字段别名，不能与同工作项类型其他字段重复 |
| field_type_key | string | 必须 | 字段的类型 |
| value_type | int64 | 可选 | 选项值来源，适用于单选、多选、级联单选、级联多选或单选按钮的情况。0-自定义、1-复用 |
| field_value | list<object> | 可选 | 选项值，支持单选、多选、级联单选、级联多选、单选按钮等结构。该属性和 team_option 互斥 |
| reference_work_item_type_key | string | 可选 | 引用字段所属工作项类型 key，当 value_type = 1 时必填 |
| reference_field_key | string | 可选 | 引用字段 ID，当 value_type = 1 时必填 |
| is_multi | boolean | 可选 | 文本字段的单行或多行标识：true-多行、false-单行 |
| format | boolean | 可选 | 日期或日期+时间标识：true-仅日期、false-日期+时间 |
| free_add | int64 | 可选 | 是否允许用户添加选项：1-是、2-否，默认2 |
| work_item_relation_uuid | string | 可选 | 工作项关系 ID，当字段类型为单选关联工作项或多选工作项时必传 |
| default_value | object | 可选 | 字段的默认值，不同字段类型的默认值结构不同 |
| help_description | string | 可选 | 帮助说明 |
| authorized_roles | list<string> | 可选 | 授权角色列表，支持角色 key 或系统人员 key |
| team_option | object | 可选 | 团队范围，只能应用于级联单选和级联多选字段。该属性和 field_value 互斥 |

返回参数

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| data | string | 字段 key |
| err | object | 请求成功时返回空值，请求失败时返回实际错误信息。 |
| err_msg | string | 请求成功时返回空值，请求失败时返回实际错误信息。 |
| err_code | int32 | 请求成功时0，请求失败返回实际错误码。 |

请求示例

请求路径：POST /open_api/64578938065xxxxx/field/story/create

```json
{
  "field_name": "openapi_create",
  "field_type_key": "text",
  "field_alias": "openapi_create",
  "help_description": "openapi_create_help_description",
  "authorized_roles": ["_master"],
  "default_value": "default value of single text"
}
```

返回示例

```json
{
  "data": "field_73jds7",
  "err": {},
  "err_msg": "",
  "err_code": 0
}
```
