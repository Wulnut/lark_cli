## Business（业务线）

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | string | 业务线 ID |
| name | string | 业务线名称 |
| project | string | 所属空间 ID |
| labels | list | 标签数组 |
| role_owners | Map<string, RoleOwner> | 默认角色及负责人，Map 的 key 为角色标识（RoleOwner.role） |
| watchers | list | 默认的关注者数组 |
| order | int64 | 排序字段 |
| super_masters | list | 业务线超级管理员数组 |
| parent | string | 父级业务线 ID |
| disabled | bool | 是否弃用 |
| level_id | int | 层级 ID，最顶层为 1，子节点层级 ID 等于父节点层级 ID+1 |

## Comment（评论）

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | int | 评论 ID |
| work_item_id | int64 | 评论所属工作项 ID |
| work_item_type_key | string | 评论所属工作项类型 |
| created_at | int | 评论创建时间，毫秒精度 |
| operator | string | 评论人 |
| content | string | 评论内容 |

## CompInfo

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | string | 工作项/视图 ID |
| name | string | 工作项/视图名称 |
| view_scope_key | string | 视图的类型（story/issue/version/sprint/chart/自定义工作项的 workitemtypekey） |
| work_item_type_key | string | 暂为空 |
| project_key | string | 所属空间 ID |
| created_by | int64 | 创建人 |
| created_at | int64 | 创建时间戳，毫秒精度 |
| search_hit | list<string> | 命中结果字段的 key |
